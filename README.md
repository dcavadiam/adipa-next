# Adipa Chile - Next.js
Esto es una landing page para la academia digital de psicología y aprendizaje Adipa Chile.

## Despliegue

Este proyecto se despliega en **Vercel**.

Sitio en producción: [https://adipa-next.vercel.app](https://adipa-next.vercel.app)


## Instrucciones de instalación en local

1. Clonar el repositorio:

```bash
git clone https://github.com/dcavadiam/adipa-next.git
cd adipa-next
```

2. Instalar dependencias (usando npm, o adapta a yarn/pnpm si prefieres):

```bash
npm install
```

## Comandos para ejecutar en desarrollo

Levantar el servidor de desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`.

## Comando de build para producción

Generar el build de producción:

```bash
npm run build
```

Opcionalmente, para probar el build localmente:

```bash
npm run start
```
